# 合资格指标要点与洞察

本文档用于沉淀 `合资格` 指标的业务理解，说明它为什么重要、为什么常见于长期经营体系、它在当前系统中的实现口径是什么，以及它适合解决什么问题、不适合单独解决什么问题。

本文档不是实现说明书，不讨论前端展示或接口设计，只讨论业务认知与指标洞察。

## 1. 什么是合资格

`合资格`本质上是一个经营门槛指标。

它回答的问题不是：

- 这个代理人是不是优秀
- 这个代理人是不是高潜
- 这个代理人的经营质量是否全面健康

它回答的是：

- 这个代理人在当前经营周期内，是否达到组织定义的最低经营线
- 如果没有达到，还差多少
- 如果达到，这种达标是稳定的还是偶发的

从经营管理角度看，`合资格`是一个非常典型的“红线指标”。

它的价值不在于描述全部真相，而在于：

- 口径清晰
- 管理动作清晰
- 可以形成持续的经营纪律

## 2. 为什么合资格是长期有效的指标

很多成熟的寿险组织长期保留 `合资格`，不是因为它全面，而是因为它非常适合作为组织管理的主轴指标。

### 2.1 它把复杂经营简化成了清晰判断

代理人经营本身非常复杂，包含：

- 新单产能
- 活动量
- 转化效率
- 客群结构
- 团队质量
- 续保质量
- 阶段性波动

如果没有一个强约束指标，管理很容易陷入“每个人都能解释、但没人真正过线”的状态。

`合资格`的作用，就是把复杂经营先收敛成一个简单问题：

```text
本期有没有达到组织认可的经营线
```

这会显著降低管理复杂度。

### 2.2 它天然适合形成经营纪律

一个好的管理指标，不只是能看，还要能反复使用。

`合资格`之所以强，是因为它天然对应以下动作：

- 月度追踪
- 季度盘点
- 经理辅导
- 晋级观察
- 人员分层
- 风险预警

也就是说，`合资格`不是纯分析指标，它是一个可以直接驱动管理动作的指标。

### 2.3 它强调“持续经营”，而不是单次爆发

好的经营体系通常不追求偶发高峰，而追求可复制、可持续的经营习惯。

`合资格`本质上是在问：

```text
你是否建立了达到组织要求的稳定经营能力
```

这比单看某个月的高业绩，更适合做组织管理。

## 3. 合资格为什么通常不是“优秀指标”，而是“门槛指标”

这是理解这个指标时最容易混淆的点。

`合资格`非常重要，但它不是为了识别最优秀的人，而是为了识别：

- 谁已经站上经营线
- 谁还没站上经营线
- 谁正在掉出经营线

所以它更像：

```text
有没有过线
```

而不是：

```text
过得有多漂亮
```

这意味着：

- 刚刚过线的人，和远超门槛的人，在 `合资格` 上可能同样是合格
- 轻微未达标的人，和严重掉队的人，在 `是否合资格` 上也可能同样是不合格

这不是这个指标的缺陷，而是这个指标的设计目的。

## 4. 当前系统中的合资格口径

当前仓库中的合资格逻辑位于 `packages/domain-performance/src/commands/batch-upsert-apm.ts`。

从现有实现看，系统中的 `合资格` 不是单一静态阈值，而是分人群、分周期计算的。

### 4.1 新人和资深代理人口径不同

当前实现将代理人按入司时长拆成两类：

- 入司未满 12 个月：按新人逻辑
- 入司满 12 个月：按资深逻辑

这说明当前系统默认接受一个重要业务判断：

```text
不同经营阶段的代理人，不应该用完全相同的合资格口径
```

这是合理的。因为新人处于起盘期，资深代理人处于稳定经营期，两者的经营要求天然不同。

### 4.2 新人口径：强调起盘节奏

新人逻辑的核心特征是：

- 以入司以来累计产能衡量
- 有明确的月度门槛
- 存在最低起步要求

这类设计的经营含义是：

- 不要求新人立即达到资深代理人的成熟节奏
- 但要求其在一个合理窗口内持续推进
- 避免新人因为单月波动被过早判死，也避免长期不起盘

换句话说，新人口径重视的是：

```text
起盘是否形成了连续推进
```

### 4.3 资深口径：强调季度经营责任

资深代理人的合资格逻辑不是单纯看某一个月的即时结果，而是结合季度目标与阶段延续来判断。

这背后的经营思想通常是：

- 资深代理人不再以“试运行”标准来看
- 组织更关心其在季度节奏上的持续责任
- 管理上允许月内波动，但不允许长期脱线

这比“每个月都重新清零考核”更符合实际经营。

### 4.4 高职级还引入了团队维度

当前实现里，较高职级代理人可以通过团队相关口径满足资格条件。

这体现了一个组织管理上的重要判断：

```text
高职级代理人的经营责任，不再只是个人产能责任，也包含团队经营责任
```

这类设计说明：

- 职级越高，越不该只看个人单兵业绩
- 更要看其是否形成团队带动能力

这也是成熟组织里很常见的资格逻辑。

## 5. 为什么合资格通常要和“差距”一起看

当前系统里不仅有 `isQualified`，还有 `qualifiedGap`。

这非常关键，因为单独看布尔值信息量太低。

例如：

- 两个人都未合资格
- A 只差很小一段
- B 差得非常远

如果只看 `是否合资格`，这两个人没有区别；  
如果看 `合资格差距`，管理动作会完全不同。

因此，正确理解应当是：

```text
合资格 = 红线状态
合资格差距 = 红线压力大小
```

一个成熟管理动作通常不是只问：

```text
你有没有合资格
```

而是会继续问：

```text
如果没有合资格，还差多少，差距主要来自哪里
```

## 6. 为什么这种设计是合理的

从管理视角看，当前系统中的合资格设计体现了几个合理原则。

### 6.1 同一组织目标，不同阶段不同口径

目标是一致的：形成稳定经营能力。  
但新人与资深、高职级与普通代理人的责任不同，因此口径不同。

这避免了两个常见问题：

- 对新人要求过高，导致管理失真
- 对资深要求过低，导致经营责任失焦

### 6.2 允许阶段波动，但不允许长期脱线

资深口径不是完全按单月即时状态处理，而是带有周期视角。

这意味着组织承认现实经营中存在波动，但同时要求在周期内完成经营责任。

这是比“单月绝对主义”更成熟的管理方式。

### 6.3 高职级必须承担组织责任

当代理人进入更高职级后，其经营意义不再只是“自己做多少”，而是“自己能否持续带出团队结果”。

这也是为什么高职级资格逻辑通常会引入团队维度。

## 7. 合资格适合解决什么问题

`合资格`非常适合以下场景：

- 经营红线管理
- 月度/季度盘点
- 经理追踪名单
- 人员分层
- 预警与复盘起点
- 晋级前置观察

如果一个经理只能先看一个指标，那么 `合资格` 往往是最合理的起点之一。

因为它会先告诉经理：

```text
现在最需要关注的人是谁
```

## 8. 合资格不适合单独解决什么问题

`合资格`不应该被过度使用。

它不适合单独回答以下问题：

- 为什么这个人掉下来了
- 这个人是活动不足，还是转化不行
- 这个人是否在恶化
- 这个人经营质量是否健康
- 这个人的产能是否可持续
- 这个人虽然合资格，但是否已经开始出现风险

也就是说：

```text
合资格适合做起点
不适合单独做终局判断
```

## 9. 对合资格指标的正确使用方式

对管理层来说，最有效的用法不是“只看合资格”，而是“以合资格为主轴，再配平行指标”。

推荐顺序是：

### 第一步：先看红线

- 是否合资格
- 当前差距多大

这一步解决：

```text
谁现在最值得被关注
```

### 第二步：再看趋势

- 最近是否连续合资格
- 最近 3 个月是在恢复还是在下滑

这一步解决：

```text
这是偶发问题，还是趋势问题
```

### 第三步：再看过程

- 活动量是否不足
- 转化效率是否下降
- 结构是否变差

这一步解决：

```text
经理下一步该谈什么
```

## 10. 合资格的典型盲区

在经营分析中，`合资格` 常见的盲区包括：

### 10.1 只看结果，不看过程

一个人不合资格，可能因为：

- 活动量不够
- 客户结构偏弱
- 转化效率下滑
- 阶段节奏失衡

`合资格` 本身不会告诉你原因。

### 10.2 只看当月，不看趋势

一个人这个月合资格，不代表没有风险。  
有可能他已经连续 3 个月走弱，只是暂时还没掉出红线。

### 10.3 只看个人，不看结构

对于较高职级，仅看个人资格可能掩盖团队问题。  
对于普通代理人，仅看资格也可能掩盖高价值业务不足、续保偏弱等问题。

### 10.4 只看布尔值，不看差距

管理上，`0/1` 只是入口，不是结论。  
真正能指导动作的是“差距有多大、变化有多快、原因在哪里”。

## 11. 关于平行指标的原则

既然 `合资格` 不够全面，那么补充指标时就不应该和它争夺主地位，而应围绕它补充视角。

一个合理的平行指标体系，通常包含以下几个方向：

- 红线压力：合资格差距、下月资格压力
- 稳定性：连续合资格月数、合资格稳定度
- 动能：近期产能趋势、下滑风险
- 过程：活动量、转化效率
- 质量：高价值业务占比、续保质量、结构健康度

这些指标的作用不是替代 `合资格`，而是回答 `合资格` 没有回答的问题。

## 12. 最终判断

关于 `合资格`，最重要的结论只有三条：

### 12.1 它是一个优秀的主红线指标

它不能代表全部经营真相，但非常适合做组织管理的起点。

### 12.2 它不是全貌指标

它适合做“有没有过线”的判断，不适合单独做“经营是否健康”的判断。

### 12.3 它应该被保留，并被平行指标补强

正确方向不是削弱 `合资格`，而是让它继续担任主轴，再补上趋势、过程、质量类指标。

可以把它概括为：

```text
合资格负责定义经营红线
平行指标负责解释红线内外发生了什么
```

## 13. 一句话总结

`合资格` 之所以长期有效，不是因为它全面，而是因为它把复杂经营压缩成了组织可以持续执行的经营纪律；它应该继续作为主红线指标存在，但必须由趋势、过程和质量指标补充，才能形成真正有洞察力的经营分析体系。
